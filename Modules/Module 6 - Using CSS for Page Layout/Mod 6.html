Module 6: Using CSS for page layout

Most web pages today are broken up into various sections. Often the navigation is on the left, the main information is in the center, and there may be a sidebar on the right. To achieve this in a web page we use columns. Chapter 6 will show you how to use CSS to format pages with 2 or 3 columns. You'll learn how to use fixed-width column layouts and fluid layouts and, most important, how to ensure that elements appear correctly, regardless of the user's screen size or resolution.

After reading this chapter and completing the exercises, you can begin to think about the type of layout you want for your final project.
