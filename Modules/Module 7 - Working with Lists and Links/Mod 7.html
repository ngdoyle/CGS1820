Module 7: Working with lists and links

Nowadays people expect information to be organized and delivered in a clear, efficient manner. Using bulleted and numbered lists is a good way to to do this. There are three main types of lists that can easily be coded with HTML: unordered lists (i.e., bulleted lists), ordered lists (i.e., numbered lists using numbers or alphabetic characters), and description lists (a bit more about this below). You can combine these types to create nested lists as well. A description list is a little different from ordered and unordered lists. It contains three elements:

  • The <dl></dl> element creates the description list.
  • The <dt></dt> element is used for the term to be defined.
  • The <dd></dd> element is used for the definition of the term.

In Chapter 7 you'll learn how to write the HTML to create all of these lists as well as how to format lists in almost any style you choose.

You have been working with links throughout the semester. The Town Hall site links to other pages on the site. The Halloween site has links in the navigation area. In this chapter you will learn to add formatting to your links (like specifying the colors you want to use when a user hovers over a link or visits a link and more). You'll also learn how to use a link to open a new browser window or tab, how to link to a media file, how to create placeholders, and how to create email, phone, and Skype links. You will also learn to add formatting or change the way your navigation menu works.

After reading this chapter and completing the exercises, you can begin to think about where you might include lists in your final project. You should also think about how you might want to format various links throughout your final project, whether you want a vertical or a horizontal navigation menu, if you want to include a multi-tier navigation menu, and so on.
