Module 10: How to work with tables

While most of the time we format our pages using CSS for layout, there are times you may want or need to add a table to your web pages. Tables aren't just used to display, for example, a table of data. By making the cells invisible it is possible to use tables to line up content in ways that may be difficult to do with CSS. And many web site "wizards" (like WordPress) are built around concepts used in tables so understanding how they work can be very helpful if you work for a company that expects you to work on one of those sites. While there are many components associated with tables (rows, columns header cells, data cells, table header, table footer, and table body), it is most important to understand how to format a table so your content displays as you want it to be seen.
