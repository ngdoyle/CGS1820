Module 4: Using CSS to format web page elements

In Module 4 you'll begin making your pages look stylish... with styles. Chapter 4 is about how web pages use CSS (Cascading Style Sheets) to alter the presentation of various items. There's so much to learn about CSS that it spans 3 chapters in the text. This week we will focus on the material in Chapter 4. You'll learn to apply different styles to various HTML elements and also how to change the styles within elements using element types, ids, classes, selectors, and more.

Of course, you should continue thinking about the website you will submit at the end of the semester for your Final Project. After you go through this chapter, you can begin thinking about a color scheme for your site.
