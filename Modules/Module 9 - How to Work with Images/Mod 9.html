Module 9: How to work with images

You've already worked with images but in this Module you'll learn more advanced techniques to use when you add images to web pages. You'll learn about the 3 types of images used with websites (.jpg, .gif, and .png) and how to use CSS to align and float images. You will also learn about a few new HTML5 elements that allow you to treat an image as a figure. And you'll learn to use an image map and to make an image into a link. You'll also learn about image rollovers and thumbnails.

The end of the semester is almost upon us so this is a good time to organize the images you plan to use on your final project. Remember: if you take images from anywhere on the Internet, you must be careful not to use someone else's work without permission or, if appropriate, giving the original owner credit.
