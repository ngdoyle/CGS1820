Module 5: Using the CSS box model

In Module 5 you'll learn to use the CSS box model to control the spacing between elements and to display borders and backgrounds. This is important when it comes to displaying a page that is readable and pleasant to look at. Take your time and go through Chapter 5 to learn some of the interesting features you can use on your pages such as rounded corners, shadows, and background gradients.

Of course, you should continue thinking about the website you will submit at the end of the semester for your Final Project. After you go through this chapter, you can begin thinking about the layout of the pages you are planning for your site.
